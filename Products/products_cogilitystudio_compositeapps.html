Event Based Workflow and Composite Applications

Applications that cross integrate and provide unique features and functions.

A Composite application goes beyond simple integration of data, by typically subsuming the operations and interfaces of other applications by providing a unified set of functions, often removing the need to directly access the integrated applications for typical tasks. For example in a telco environment a composite application might integrate the billing, order entry, and workforce management functions into one integrated customer care and self service web interface. This would require that the new application not just exchange data with the existing systems, but add new functions such as the custom self service portal, and often means having integrated interfaces for call center agents that would in the past be accessing each system separately.
